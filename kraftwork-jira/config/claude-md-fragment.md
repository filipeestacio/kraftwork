## Ticket Management (Jira)

- Use `ticket-management-describe` to read the full ticket — acceptance criteria, description, and comments — before starting work.
- Use `ticket-management-find` to search for related tickets before creating new ones.
- Update ticket status with `ticket-management-update` when starting work and when submitting for review.
- If a ticket is unclear, add a comment via `ticket-management-update` rather than assuming intent.
