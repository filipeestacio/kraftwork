## Ticket Management & Documents (ClickUp)

- Use `ticket-management-describe` to read the full task — description, custom fields, and comments — before starting work.
- Use `ticket-management-find` to search for related tasks before creating new ones.
- Update task status with `ticket-management-update` when starting work and when submitting for review.
- Use `document-storage-find` and `document-storage-describe` to check for existing documentation before writing new docs.
- Save specs and decision docs via `document-storage-create` so they are accessible to the whole team.
